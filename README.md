# Jain Solar Quotation Generator

A professional React application for generating solar energy system quotations with PDF export functionality.

## Features

- ✅ Interactive form-based quote builder
- ✅ Real-time pricing calculations with GST
- ✅ Residential subsidy support (PM Surya Ghar Yojna)
- ✅ Professional 3-page PDF generation
- ✅ Print-optimized layouts
- ✅ Responsive design with Tailwind CSS

## Technologies Used

- **React 18** - UI Framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
- **html2pdf.js** - PDF generation

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Usage

1. **Edit Mode**: Fill in customer details, system specifications, and pricing
2. **Preview Mode**: Review the professional 3-page quotation
3. **Export**: Download as PDF or print directly
